Privacy Policy

Developed by ahmadafif5321.

whisper-malay v2 records speech, transcribes it, and inserts the result into focused Android text fields after you tap the overlay button.

Local mode

Audio is processed on-device using local ASR models. Audio does not leave the phone.

Cloud mode

Audio is sent directly from the device to Google Gemini using your own API key. Optional cleanup sends transcript text directly to Gemini and is designed to preserve Bahasa Melayu, Manglish, and particles.

Accessibility Service

The Accessibility Service is used only to identify the focused text field and insert dictated text after explicit overlay interaction. It is not for analytics, browsing monitoring, keylogging, or background automation.

Data collection

This project does not collect accounts, analytics, recordings, or transcripts, and it does not operate a relay backend.
