whisper-malay v2

Push-to-talk dictation for Android, tuned around Bahasa Melayu Malaysia, Manglish, and practical cross-app text insertion.

  • Tap the overlay, speak, tap again, and insert text into the focused field.
  • Use local Whisper-compatible models or Gemini cloud transcription.
  • Malay readiness checks show whether the selected model supports ms.
  • Hugging Face direct archive URLs can be downloaded into local models.

Malay-first model flow

The app prompts for Malay Whisper Small on first launch. Malay Whisper Medium stays available as the best-quality option with a storage/RAM warning.

Recommended

  • Malay Whisper Small
  • Good local quality for Bahasa Melayu
  • Safer default than the Medium model

Best quality

  • Malay Whisper Medium
  • More accurate, slower, and much larger
  • Shows a warning before download

Custom imports

  • Import local archives from storage
  • Pull direct Hugging Face /resolve/ zip URLs
  • Archives are validated before installation

Private by default

Local mode keeps audio on-device. Cloud mode sends audio directly from your phone to Gemini using your own API key. This project runs no relay backend.
